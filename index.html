Algunos lenguajes de la web

HTML+CSS+Javascrip

  • HTML 5

    • HTML 5(Hiper Text Markup Lenguaje,versión 5)es la quinta revisión importante del lenguaje basico de la World Wide Web,HTML.

    Usa la extención.html

  • CSS 3

    • Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets)es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.

    Usa la extención .css

  • JavaScript

    • JavaScript(abreviado comunmente "JS")es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (cliente-side),implementando como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas.

    Usa la extención.js
